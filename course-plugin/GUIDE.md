# Руководство по использованию плагина "Курсы"

## 📍 Где найти плагин в WordPress

### 1. Активация плагина

**Путь:** `Админ-панель → Плагины → Установленные плагины`

После установки плагина в папку `wp-content/plugins/`:
1. Найдите в списке плагин **"Курсы"**
2. Нажмите кнопку **"Активировать"**

---

### 2. Главное меню в админ-панели

После активации в **левом боковом меню** появится новый пункт:

```
📚 Курсы
   ├─ Все курсы
   ├─ Добавить новый
   ├─ Специализации и программы
   ├─ Уровни образования
   ├─ Темы
   └─ Преподаватели
```

**Позиция:** Между "Записи" и "Медиафайлы" (позиция 5)  
**Иконка:** 📚 (книга)

---

### 5. Управление таксономиями

#### Специализации и программы
**Путь:** `Курсы → Специализации и программы`

- Можно создавать иерархические категории
- Пример: Программирование → Веб-разработка → Backend

#### Уровни образования
**Путь:** `Курсы → Уровни образования`

- Иерархическая структура
- Пример: Высшее образование → Бакалавриат

#### Темы
**Путь:** `Курсы → Темы`

- Иерархическая структура
- Пример: Программирование → Языки → PHP

#### Преподаватели
**Путь:** `Курсы → Преподаватели`

- Теги (неиерархические)
- Можно добавить несколько преподавателей к одному курсу

---

### 6. На фронтенде сайта

#### Архив курсов
**URL:** `ваш-сайт.ru/courses/`

Отображает список всех опубликованных курсов с:
- Миниатюрами
- Названиями
- Кратким описанием
- Ссылками на полные страницы курсов

#### Страница отдельного курса
**URL:** `ваш-сайт.ru/courses/название-курса/`

Отображает:
- Полное описание курса
- Изображение курса
- Информацию о таксономиях
- Дополнительные поля (стоимость, продолжительность и т.д.)

---

### 7. Быстрый старт

1. **Активируйте плагин** через меню "Плагины"
2. **Создайте таксономии:**
   - Добавьте несколько специализаций
   - Добавьте уровни образования
   - Добавьте темы
   - Добавьте преподавателей
3. **Создайте первый курс:**
   - `Курсы → Добавить новый`
   - Заполните заголовок и описание
   - Выберите таксономии
   - Заполните дополнительные поля
   - Опубликуйте
4. **Проверьте на сайте:**
   - Перейдите на `/courses/` чтобы увидеть архив
   - Откройте созданный курс

---

### 8. Полезные советы

- ✅ Используйте фильтры в списке курсов для быстрого поиска
- ✅ Заполняйте все поля для лучшей организации
- ✅ Используйте изображения курсов (миниатюры)
- ✅ Таксономии можно редактировать прямо при создании курса
- ✅ Все курсы доступны через REST API: `/wp-json/wp/v2/courses`

---

## 🔧 Технические детали

- **Custom Post Type:** `course`
- **Таксономии:**
  - `course_specialization` (иерархическая)
  - `course_level` (иерархическая)
  - `course_topic` (иерархическая)
  - `course_teacher` (неиерархическая)
- **Метаполя:**
  - `_course_duration` - продолжительность в часах
  - `_course_price` - стоимость
  - `_course_start_date` - дата начала
  - `_course_end_date` - дата окончания
  - `_course_capacity` - вместимость
  - `_course_enrolled` - количество записанных
