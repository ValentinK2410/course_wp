# Плагин "Курсы Про" для WordPress

Плагин для управления курсами в WordPress с возможностью добавления, редактирования и удаления курсов.

## Возможности

- **Custom Post Type "Курсы"** - полноценный тип записей для курсов
- **Таксономии:**
  - Специализация и программы
  - Уровень образования
  - Тема
  - Преподаватель
- **Дополнительные поля:**
  - Продолжительность курса (в часах)
  - Стоимость курса
  - Дата начала и окончания
  - Вместимость (количество мест)
  - Количество записанных студентов
- **Административный интерфейс:**
  - Колонки в списке курсов с информацией о таксономиях
  - Фильтры по таксономиям
  - Сортировка по таксономиям

## Установка

1. Скопируйте папку `course-plugin` в директорию `wp-content/plugins/` вашего WordPress сайта
2. Активируйте плагин через меню "Плагины" в админ-панели WordPress
3. После активации в меню появится пункт "Курсы Про"

## Использование

### Добавление курса

1. Перейдите в меню "Курсы Про" → "Добавить новый"
2. Заполните заголовок и описание курса
3. Выберите или создайте термины для таксономий:
   - Специализация и программы
   - Уровень образования
   - Тема
   - Преподаватель
4. Заполните дополнительные поля в метабоксах
5. Нажмите "Опубликовать"

### Редактирование курса

1. Перейдите в меню "Курсы Про" → "Все курсы"
2. Нажмите на нужный курс для редактирования
3. Внесите изменения и нажмите "Обновить"

### Удаление курса

1. Перейдите в меню "Курсы Про" → "Все курсы"
2. Наведите курсор на нужный курс
3. Нажмите "Удалить" или переместите в корзину

### Управление таксономиями

Таксономии можно управлять через боковую панель при редактировании курса или через отдельные разделы в меню "Курсы Про".

### Шорткоды для главной страницы и других страниц

Плагин поддерживает шорткоды для вывода курсов, программ и преподавателей с указанием количества:

**Курсы** — `[courses]`
```
[courses per_page="6"]
[courses per_page="12" specialization="bogoslovie"]
[courses per_page="8" level="bachelor" view="grid"]
```
Параметры: `per_page` (по умолчанию 12), `specialization`, `level`, `topic`, `teacher`, `view` (grid/list)

**Программы** — `[programs]`
```
[programs per_page="4"]
[programs per_page="8" specialization="bogoslovie"]
```
Параметры: `per_page` (по умолчанию 12), `specialization`, `level`, `topic`, `teacher`, `view`

**Преподаватели** — `[teachers]`
```
[teachers per_page="6"]
[teachers per_page="8" columns="4"]
```
Параметры: `per_page` (по умолчанию 6), `columns` (2–4, по умолчанию 3)

**Пример для главной страницы:**
```
[courses per_page="3"]

[teachers per_page="4"]

[programs per_page="2"]
```
